# 本地开发速查

## 启动方式

```bash
# 1. 后端
cd backend && source venv/bin/activate && uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 2. 前端
cd frontend && npm install && npm run dev   # → http://localhost:5173/
```

## 访问地址

| 目标 | URL | 说明 |
|------|-----|------|
| **前台首页** | `http://localhost:5173/#/` | 公众页面入口 |
| **管理后台登录** | `http://localhost:5173/#/admin/login` | admin / admin123 |
| **报名** | `http://localhost:5173/#/register` | |
| **作品展示** | `http://localhost:5173/#/showcase` | |
| **人气投票** | `http://localhost:5173/#/vote` | |
| **API 文档** | `http://localhost:8000/docs` | Swagger |

## ⚠️ HashRouter

前端使用 `HashRouter`，所有路由在 `#` 之后：

- ❌ `http://localhost:5173/admin/login` → 错误
- ✅ `http://localhost:5173/#/admin/login` → 正确

## 测试

```bash
cd frontend
npm test           # 82 个测试
```

## 构建

```bash
cd frontend
npm run build      # base: '/' (适用于 Railway)
npx vite build --base /zhonghui-competition/  # 适用于 GitHub Pages
```
