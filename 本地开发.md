# 本地开发速查

## 启动方式

```bash
# 1. 后端
cd backend && source venv/bin/activate && uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 2. 前端（可选，后端已托管静态文件）
cd frontend && npm run dev   # → http://localhost:5173/
```

## 访问地址

| 目标 | URL | 说明 |
|------|-----|------|
| **管理后台登录** | `http://localhost:8000/#/admin/login` | admin / admin123 |
| **管理后台主页** | `http://localhost:8000/#/admin/review` | 审核管理页 |
| **前台首页** | `http://localhost:8000/#/` | 公众页面入口 |
| **报名** | `http://localhost:8000/#/register` | |
| **作品展示** | `http://localhost:8000/#/showcase` | |
| **人气投票** | `http://localhost:8000/#/vote` | |
| **API 文档** | `http://localhost:8000/docs` | Swagger |

## ⚠️ 关键：HashRouter

前端使用 `HashRouter`，所有路由在 `#` 之后：

- ❌ `http://localhost:8000/admin/login` → 错误，会跳转到 `#/auth`
- ✅ `http://localhost:8000/#/admin/login` → 正确
