## 部署流程

```bash
# 1. 推送代码到 GitHub
git add . && git commit -m "update"
git push origin master

# 2. Railway 自动部署（或通过 CLI 手动触发）
railway up --detach
```

Docker 构建流程：安装 Node.js → `npm ci` → `npm run build -- --base /static/` → 安装 Python 依赖 → 复制后端代码 → 将前端 dist 放入 `/app/static` → 启动 uvicorn。

当前仓库中的 `startup.sh` 仅负责启动 `uvicorn`。数据库迁移（`alembic upgrade head`）和种子初始化（`python -m app.utils.seed`）需在部署前或运维流程中手动完成。
