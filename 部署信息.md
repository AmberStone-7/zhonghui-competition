## 部署流程

### 前端 → GitHub Pages

```bash
cd frontend
npx vite build --base /zhonghui-competition/
# 将 dist/ 内容推送到 gh-pages 分支
```

### 后端 → Railway

详见 [docs/部署避坑指南.md](docs/部署避坑指南.md)。

## Railway 环境变量

| 变量 | 说明 |
|------|------|
| DATABASE_URL | Supabase PostgreSQL 连接串 |
| SUPABASE_URL / SUPABASE_KEY | Supabase 配置 |
| JWT_SECRET_KEY | JWT 签名密钥 |
| FRONTEND_URL | https://amberstone-7.github.io/zhonghui-competition |
