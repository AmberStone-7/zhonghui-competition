## 部署流程

```bash
# 1. 推送代码到 GitHub
git add . && git commit -m "update" && git push origin main

# 2. Railway 自动部署（或通过 CLI 手动触发）
railway up --detach
```

Docker 构建流程：安装 Node.js → 编译前端 → 安装 Python 依赖 → 复制后端代码 → 将前端 dist 放入 static/ → 启动服务。

启动时自动执行数据库迁移（alembic upgrade head），首次部署还会执行种子数据初始化。
